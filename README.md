# HabitFlow – Dariel's Habit Tracker

A custom habit tracking dashboard built with HTML, CSS & JavaScript.
Tracks 11 daily habits with streaks, goals, weekly to-dos, and a progress circle.

## Features
- 11 habits: Wasser, Spazieren, Laufen, Sport, Mobility, Stretching, 8h Schlaf, Vitamine, Kreatin, Skincare, Aufräumen
- Monthly calendar grid with checkboxes
- Streak tracking & goal progress
- Weekly to-do lists (5 weeks × 10 items)
- Activities feed & water glass tracker
- All data saved in browser localStorage

## Deploy
Deployed via Vercel at https://habitflow.vercel.app
